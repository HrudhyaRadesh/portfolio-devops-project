About This Project

A practical DevOps project demonstrating automated deployment of a website.

Project Description

This project is a multi-page portfolio website built using HTML, CSS, and JavaScript. The main objective of the project is to demonstrate DevOps automation using Git, Jenkins, and Docker.

The website source code is stored in GitHub. Jenkins pulls the latest code from the GitHub repository, builds a Docker image using the Dockerfile, removes the old container, and deploys the updated website in a new Docker container.

Why This Project Is Useful

  • Shows practical understanding of CI/CD pipeline.
  • Demonstrates automated deployment using Jenkins.
  • Uses Docker for consistent containerized deployment.
  • Uses GitHub for source code management.
  • Can be easily extended and deployed to cloud platforms.
