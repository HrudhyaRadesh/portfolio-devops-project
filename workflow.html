CI/CD Workflow

Step-by-step automated deployment workflow used in this project.

1. Code Development

Website files are created using HTML, CSS, and JavaScript in VS Code.

2. Git Commit

Changes are tracked using Git and committed with meaningful messages.

3. GitHub Push

Source code is pushed to GitHub remote repository.

4. Jenkins Pipeline

Jenkins pulls the latest code and executes the Jenkinsfile stages.

5. Docker Image Build

Docker builds an image using the Nginx base image and website files.

6. Container Deployment

Old container is removed and a new updated container is deployed.
